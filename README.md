# practice-vue

## Project Description

本專案基於現有網頁進行前端技術練習。
使用技術有:

- _HTML_
- _CSS_
- _Javascript_
- _Typescript_
- _Sass_
- _Vuejs (Vue, Vue-Router)_
- _Axios_
- _i18next_
- _Swiper_
- _PostCSS_
- _MSW_
- _Vitejs_

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
